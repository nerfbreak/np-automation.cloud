# STRICT NEWSPAGE BOT RULES

1. ALWAYS strictly follow the exact Element IDs and code snippets provided by the user.
2. DO NOT BE 'SOTOY' (presumptuous/know-it-all). Never invent or assume the existence of elements (like checkboxes, dropdowns, etc.) that the user did not explicitly mention or provide code for.
3. Keep actions strictly limited to what the user explicitly requested.
